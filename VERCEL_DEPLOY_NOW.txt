================================================================================
TROUBLESHOOTING
================================================================================

❌ Deployment fails?

Check build logs:
→ Vercel dashboard → Your project → Deployments tab
→ Click latest deployment → View Build Logs
→ Look for error message

Common errors:

A) "No module named 'flask'"
   Fix: Make sure requirements.txt is in project root ✓

B) "model.pkl not found"
   Fix: Model files are already pushed ✓ (should work)

C) "Port error"
   Fix: Our config handles this ✓ (Vercel auto-uses dynamic port)
