================================================================================
COMMON ERRORS & FIXES
================================================================================

ERROR: "No module named 'flask'"
→ Run: pip install flask


ERROR: "loan_data.csv not found"
→ Make sure loan_data.csv is in the project root folder


ERROR: "model.pkl not found"
→ Run: python train_model.py (FIRST)


ERROR: "Address already in use"
→ The port 5000 is busy. Try:
   python app.py (it will use a different port)
   Or wait a minute and try again


ERROR: "Connection refused"
→ Make sure Flask server is running (see Step 4)
→ Make sure you're using correct URL: http://localhost:5000


ERROR: Form not submitting
→ Fill ALL fields (marked with *)
→ Check browser console: F12 → Console tab
→ Make sure Flask server is still running
