================================================================================
TROUBLESHOOTING
================================================================================

❌ Git push fails?
→ Update git credentials at: https://github.com/settings/tokens
→ Create "Personal Access Token" and use as password

❌ Vercel deployment fails?
→ Check build logs in Vercel dashboard
→ Make sure requirements.txt is in project root
→ All .pkl files must be in project root

❌ App not working after deployment?
→ Check Vercel Runtime Logs
→ Verify all imports in app.py work
→ Check if model files uploaded properly
